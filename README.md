# Todo App


### Description:  
This app allows users to log in and create todo itmes.  Once created the items can be deleated, edited, and marked complete.

### To run:
```
$ cd lib /
$ npm install
$ bower install
$ gulp
$ cd ..
$ http-server -p 9999
```

Your app should now be running on [localhost:9999](http://localhost:9999/).
